## About Project<br>
코로나가 오랫동안 지속되면서 힘들어하는 소상공인들을 도울 수 있는 웹 페이지를 만들고 싶어서 BBira를 기획하게 되었습니다.

사업자가 주력 상품을 쉽게 등록 할 수 있는 기능이 있기 때문에 소비자에 다양한 상품을 보다 쉽게 홍보 할 수 있게 해줍니다!
